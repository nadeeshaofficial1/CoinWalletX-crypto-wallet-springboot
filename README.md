## Architecture

CoinWalletX is designed with a microservices architecture. Here's a high-level overview:

- **Frontend:** React JS
- **Backend:** Java Spring Boot
- **Database:** MySQL
- **Microservices:** Independent services for coin management, payments, and user interactions
